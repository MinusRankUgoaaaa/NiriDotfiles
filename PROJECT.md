- Performance
- Gaming
- Simplicity
- Maintainability

## Design

- Minimalistic
- No shadows
- Slight transparency
- Small corner radius
- Fast animations
- Dynamic colors (optional)
- Static theme support

## Repository Philosophy

Repository is the single source of truth.

Nothing is configured manually inside ~/.config.

Everything is linked using bootstrap.sh.
